  at 50 requests per 10 seconds per IP. Higher bandwidths will soon become
  available via the public network.
</Info>

This section lists public EVM gateways available from the [open private network](en/subsquid-network/overview/#open-private-network). If you are developing a squid, supply the gateway URL of your network to [`setGateway()`](en/sdk/reference/processors/evm-batch/general/#set-gateway) `EvmBatchProcessor` configuration method, for example:

```typescript
const processor = new EvmBatchProcessor().setGateway(
  "https://v2.archive.subsquid.io/network/ethereum-mainnet"
);
```

See [EVM API](en/subsquid-network/reference/evm-api) if you're looking to interface with one of these gateways directly.

## L1 fields availability

Extra information on L1 [height](en/sdk/reference/processors/evm-batch/field-selection/#block-headers) and [transaction fees](en/sdk/reference/processors/evm-batch/field-selection/#transactions) is available for select Layer 2 EVM networks:

- Optimism
- Optimism Sepolia
- Base
- Base Sepolia
- Zora
- Zora Sepolia
- Scroll (`l1Fee` only)
- Scroll Sepolia (`l1Fee` only)
