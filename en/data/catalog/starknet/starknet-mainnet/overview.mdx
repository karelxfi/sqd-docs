## Network Information

- **Network ID**: `starknet-mainnet`
- **Gateway**: Open Private Network

## Endpoints

### Gateway Endpoint

```
https://v2.archive.subsquid.io/network/starknet-mainnet
```

### Usage

```typescript
import { StarknetDataSource } from '@subsquid/starknet-stream'

const dataSource = new StarknetDataSource()
  .setGateway("https://v2.archive.subsquid.io/network/starknet-mainnet")
  .setBlockRange({ from: 0 });
```
