  available via the public network.
</Info>

## Network Information

- **Network ID**: `soon-mainnet`
- **Gateway**: Open Private Network

## Endpoints

### Gateway Endpoint

```
https://v2.archive.subsquid.io/network/soon-mainnet
```

### Usage

```typescript
import { SolanaDataSource } from '@subsquid/solana-stream'

const dataSource = new SolanaDataSource()
  .setGateway("https://v2.archive.subsquid.io/network/soon-mainnet")
  .setBlockRange({ from: 0 });
```
