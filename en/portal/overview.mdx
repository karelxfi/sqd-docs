## How It Works

SQD Portal provides a simple HTTP API for querying blockchain data:

1. **Make HTTP POST requests** with your desired block range and filters
2. **Receive streaming responses** with newline-delimited JSON
3. **Handle reorgs automatically** with built-in finality tracking
