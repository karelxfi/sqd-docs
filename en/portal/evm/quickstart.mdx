Make your first Portal request to access raw EVM blockchain data in under 5 minutes. No setup required—just HTTP requests.

## What You'll Build

In this quickstart, you'll:

1. Make a simple Portal request using curl (no installation required)
2. Query recent Ethereum blocks
3. Filter event logs from a smart contract

<Info>
  This guide uses the **Public Portal** endpoint, which is free and
  rate-limited. Perfect for getting started.
</Info>

## Step 1: Your First Request

Let's query the last 1,000 blocks from Ethereum mainnet. Copy and paste this command:

<Tabs>
<Tab title="curl">
```bash
curl -X POST 'https://portal.sqd.dev/datasets/ethereum-mainnet/stream' \
  -H 'Content-Type: application/json' \
  -d '{
    "type": "evm",
    "fromBlock": 18000000,
    "toBlock": 18001000,
    "fields": {
      "block": {
        "number": true,
        "timestamp": true,
        "gasUsed": true
      }
    }
  }'
```

<Check>
You should see a stream of JSON objects, one per block, showing block numbers, timestamps, and gas usage.
</Check>
</Tab>

<Tab title="TypeScript">
First, install the client:

```bash
npm install @subsquid/portal-client
```

Then run this code:

```typescript
import { DataSource } from "@subsquid/portal-client";

const dataSource = new DataSource({
  network: "ethereum-mainnet",
});

const blocks = await dataSource.getBlocks({
  from: 18000000,
  to: 18001000,
  fields: {
    block: {
      number: true,
      timestamp: true,
      gasUsed: true,
    },
  },
});

console.log(`Retrieved ${blocks.length} blocks`);
blocks.forEach((block) => {
  console.log(`Block ${block.header.number}: ${block.header.gasUsed} gas used`);
});
```

</Tab>

<Tab title="Python">
First, install requests:

```bash
pip install requests
```

Then run this code:

```python
import requests
import json

url = "https://portal.sqd.dev/datasets/ethereum-mainnet/stream"
headers = {"Content-Type": "application/json"}
payload = {
    "type": "evm",
    "fromBlock": 18000000,
    "toBlock": 18001000,
    "fields": {
        "block": {
            "number": True,
            "timestamp": True,
            "gasUsed": True
        }
    }
}

response = requests.post(url, headers=headers, json=payload)

# Response is newline-delimited JSON
for line in response.text.strip().split('\n'):
    block = json.loads(line)
    print(f"Block {block['header']['number']}: {block['header']['gasUsed']} gas used")
```

</Tab>
</Tabs>

## Step 2: Understanding the Request

Let's break down what you just sent:

```typescript
{
  "type": "evm",              // Chain type (EVM, Solana, etc.)
  "fromBlock": 18000000,      // Starting block (inclusive)
  "toBlock": 18001000,        // Ending block (inclusive)
  "fields": {                 // What data to return
    "block": {
      "number": true,         // Block number
      "timestamp": true,      // Block timestamp
      "gasUsed": true         // Total gas used in block
    }
  }
}
```

<Tip>
  Portal only returns the fields you request. This keeps responses fast and
  bandwidth-efficient.
</Tip>

## Step 3: Understanding the Response

Portal returns **newline-delimited JSON (NDJSON)**. Each line is a complete JSON object representing one block:

```json
{"header": {"number": 18000000, "timestamp": 1697544779, "gasUsed": "12534567"}, "transactions": [], "logs": []}
{"header": {"number": 18000001, "timestamp": 1697544791, "gasUsed": "14234891"}, "transactions": [], "logs": []}
{"header": {"number": 18000002, "timestamp": 1697544803, "gasUsed": "13891234"}, "transactions": [], "logs": []}
```

<Note>
  This format enables **constant-memory streaming** of massive ranges. You can
  process millions of blocks without loading everything into RAM.
</Note>

## Step 4: Filter Event Logs

Track USDC Transfer events:

<CodeGroup>
```bash curl
curl -X POST 'https://portal.sqd.dev/datasets/ethereum-mainnet/stream' \
  -H 'Content-Type: application/json' \
  -d '{
    "type": "evm",
    "fromBlock": 18000000,
    "toBlock": 18010000,
    "fields": {
      "log": {
        "address": true,
        "topics": true,
        "data": true,
        "transactionHash": true
      }
    },
    "logs": [{
      "address": ["0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48"],
      "topic0": ["0xddf252ad1be2c89b69c2b068fc378daa952ba7f163c4a11628f55a4df523b3ef"]
    }]
  }'
```

```typescript TypeScript
import { DataSource } from "@subsquid/portal-client";

const dataSource = new DataSource({
  network: "ethereum-mainnet",
});

const blocks = await dataSource.getBlocks({
  from: 18000000,
  to: 18010000,
  fields: {
    log: {
      address: true,
      topics: true,
      data: true,
      transactionHash: true,
    },
  },
  logs: [
    {
      address: ["0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48"], // USDC
      topic0: [
        "0xddf252ad1be2c89b69c2b068fc378daa952ba7f163c4a11628f55a4df523b3ef",
      ], // Transfer
    },
  ],
});

for (const block of blocks) {
  for (const log of block.logs) {
    console.log(`Transfer in tx ${log.transactionHash}`);
  }
}
```

```python Python
import requests
import json

url = "https://portal.sqd.dev/datasets/ethereum-mainnet/stream"
headers = {"Content-Type": "application/json"}
payload = {
    "type": "evm",
    "fromBlock": 18000000,
    "toBlock": 18010000,
    "fields": {
        "log": {
            "address": True,
            "topics": True,
            "data": True,
            "transactionHash": True
        }
    },
    "logs": [{
        "address": ["0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48"],
        "topic0": ["0xddf252ad1be2c89b69c2b068fc378daa952ba7f163c4a11628f55a4df523b3ef"]
    }]
}

response = requests.post(url, headers=headers, json=payload)

for line in response.text.strip().split('\n'):
    block = json.loads(line)
    for log in block.get('logs', []):
        print(f"Transfer in tx {log['transactionHash']}")
```

</CodeGroup>

<Check>
  You should see Transfer events from the USDC contract. Portal filtered the
  data before sending, saving bandwidth.
</Check>
