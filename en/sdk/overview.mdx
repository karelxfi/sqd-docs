SQD offers three ways to access blockchain data. Choose the one that fits your needs.

<CardGroup cols={3}>
  <Card title="Portal API" icon="network-wired" href="/en/portal/evm/quickstart">
    Direct HTTP access to raw blockchain data. Query using simple HTTP requests in any programming language.

**Key features:**
- Language-agnostic HTTP API
- 10-50x faster than RPC
- Minimal setup required
- Full control over processing

**Best for:** Analytics, data lakes, custom pipelines in any language
  </Card>

  <Card title="Pipes SDK" icon="bars-staggered" href="/en/sdk/pipes-sdk/quickstart">
    Lightweight TypeScript streaming library for custom data pipelines with automatic decoding and type safety.

**Key features:**
- Flexible streaming architecture
- Use any database or target
- Full control over data flow
- Minimal dependencies

**Best for:** Custom pipelines, existing codebases, non-PostgreSQL databases
  </Card>

  <Card title="Squid SDK" icon="server" href="/en/sdk/squid-sdk/evm/quickstart">
    Complete TypeScript framework with built-in PostgreSQL and auto-generated GraphQL API.

**Key features:**
- Built-in PostgreSQL + GraphQL
- Auto-generated types and migrations
- CLI tools and scaffolding
- Cloud deployment ready

**Best for:** dApp backends, GraphQL APIs, rapid development
  </Card>
</CardGroup>
