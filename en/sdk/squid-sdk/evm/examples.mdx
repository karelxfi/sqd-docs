## By Use Case

### DeFi Protocols

- **DEX Trading**: Index decentralized exchange activity
- **Lending Protocols**: Track lending and borrowing events
- **Yield Farming**: Monitor liquidity provision and rewards

### NFT & Gaming

- **NFT Collections**: Index minting, transfers, and sales
- **Gaming Assets**: Track in-game item transfers
- **Marketplace Activity**: Monitor NFT marketplace events

### Governance & DAOs

- **Voting Systems**: Index governance proposals and votes
- **Treasury Management**: Track DAO treasury movements
- **Token Distribution**: Monitor airdrops and distributions

## Using Templates

All examples are available as templates that you can scaffold using Squid CLI:

```bash
sqd init my-indexer -t <template-url>
```

<Note>
Replace `<template-url>` with the GitHub URL of any example repository to create a new project based on that template.
</Note>

<Check>
  After initializing, run `npm install` and follow the README instructions to
  get started.
</Check>
