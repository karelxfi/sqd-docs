# Quickstart

This 5-minute tutorial shows you how to build a blockchain data pipeline using Pipes SDK. You'll create a simple pipeline that fetches USDC transfer events from Ethereum and displays them in your console.

## What You'll Build

Your pipeline will:

- Fetch USDC Transfer events from Ethereum block 20,000,000
- Display the transfer data in your console
- Demonstrate the basic Source â†’ Target pattern

## Prerequisites

Before you begin, ensure you have:

- **Node.js v18+** - [Download here](https://nodejs.org/)
- **npm, yarn, or Bun** - Package manager of your choice
- **Basic TypeScript knowledge** - Familiarity with TypeScript syntax

<Steps>

Create a new directory and initialize your project:

<CodeGroup>
```bash npm

cd my-first-pipeline
npm init -y
npm install @sqd-pipes/pipes @sqd-pipes/evm-processor
```

```bash yarn

cd my-first-pipeline
yarn init -y
yarn add @sqd-pipes/pipes @sqd-pipes/evm-processor
```


<Step title="Create your first pipeline">

Create a file called `index.ts` with the following code:

```typescript index.ts
import { createEvmSource } from "@sqd-pipes/evm-processor";
import { createTarget } from "@sqd-pipes/pipes";

// Define the USDC contract address
const USDC_CONTRACT = "0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48";

// Create a data source
const source = createEvmSource({
  gateway: "https://v2.archive.subsquid.io/network/ethereum-mainnet",
  chain: "ethereum-mainnet",
  query: {
    range: {
      from: 20_000_000,
      to: 20_000_000,
    },
    logs: [
      {
        address: [USDC_CONTRACT],
        topic0: [
          "0xddf252ad1be2c89b69c2b068fc378daa952ba7f163c4a11628f55a4df523b3ef",
        ], // Transfer event signature
      },
    ],
  },
});

// Create a console target
const target = createTarget({
  write: async (data) => {
    console.log("Received data:");
    console.log(JSON.stringify(data, null, 2));
  },
});

// Run the pipeline
async function main() {
  for await (const batch of source.read()) {
    await target.write(batch);
  }
}

main().catch(console.error);
```

<Info>
  This pipeline fetches all Transfer events from the USDC contract at Ethereum
  block 20,000,000 and prints them to the console.
</Info>

</Step>

## Understanding the Code

Let's break down what each part does:

### Data Source

```typescript
const source = createEvmSource({
  gateway: 'https://v2.archive.subsquid.io/network/ethereum-mainnet',
  chain: 'ethereum-mainnet',
  query: { ... }
})
```

The source connects to the SQD Network gateway and specifies:

- Which blockchain to query (`ethereum-mainnet`)
- What block range to fetch (`20_000_000` to `20_000_000`)
- What event logs to retrieve (USDC Transfer events)

### Target

```typescript
const target = createTarget({
  write: async (data) => {
    console.log(JSON.stringify(data, null, 2));
  },
});
```

The target defines what to do with the data. In this case, it simply logs the data to the console.

### Pipeline Execution

```typescript
for await (const batch of source.read()) {
  await target.write(batch);
}
```

This loop reads data batches from the source and writes them to the target.
